Comandos git

  • git clone: clona um repósitorio do github já existente para nossa máquina.
  • git add: adiciona elementos criados ou modificados e os deixão prontos para serem comitados
  • git status: verifica o status dos arquivos dentro do nosso repósitorio informando arquivos que foram criados ou modificados
  • git commit -m "": cria uma nova versão com uma mensagem para arquivos que foram modificados ou adicionados
  • git push origin main: enviar as alterações comitadas para a branch main
  • git branch: lista todas nossas branch atual, se quisermos adicionar uma nova branch passamos o nome dela depois do comando
  • git checkout: Utilizado para andar entre as branchs ou commits, é necessário passar o nome da branch ou identificador do commit após o comando
